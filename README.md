## 📖 Project Overview / Summary
> 🔝 [Back to TOC](#-table-of-contents)

This program is a future value calculator that performs calculations using the data that is inserted by the user. It also serves as retirement countdown counter that takes your retirement date and counts how many years are left before then. The output box centered at the bottom outputs a "live projection" that updates every second, showing how your money grows over each year.*

# 💻Required Inputs:
Field | What is it?
------------- | -------------
Name | User's Name
Email | User's Email
Current Savings💲| Current balance of savings
Monthly Contribution | Monthly deposit or withdrawal from retirement plan
Estimated Rate (%): | Rate of growth 
Retirement Date | Planned retirement date
